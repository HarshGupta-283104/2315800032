# Logging Middleware

## Features

- Logs incoming requests
- Logs HTTP methods
- Logs endpoint URLs
- Logs response status codes
- Logs execution time
- Stores logs in logs.txt

## Example Log

2026-06-10T10:00:00.000Z - GET /notifications | Status: 200 | Duration: 12ms

## Run

```bash
npm install express
npm start
```
